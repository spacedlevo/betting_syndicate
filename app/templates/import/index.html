Import Transactions

Import historical transactions from a CSV file. This will create a new season, players, and all ledger entries.

CSV Format: Date, Player, Amount, Transaction
Date format: DD/MM/YYYY
Transaction types: paid in, placed, won, paid out
Set an end date to mark this as a completed/frozen season
Leave unchecked for historical seasons

Import Week Assignments

Import week assignments (betting calendar) from a CSV file into an existing season.

CSV Format: start_date, player1_name, player2_name
Date format: DD/MM/YYYY
Note: Each row represents one week
